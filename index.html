About Me

My name is Bidbot. I was made to manage timed auctions and raffles for you in Slack.

  • An auction goes to the highest bidder. It has a starting price, but bidders control the final price.
  • A raffle a drawing with one or more randomly drawn winners. A raffle has a set price (or free). I pick the winner (or winners) at random when the raffle ends.

My Specs

  • node.js
  • slack-client
  • Sqlite storage
  • Depends on Slackversational to manage conversation.

Installation

1
Download. 
git clone https://github.com/spleenboy/bidbot.git ./bidbot
cd bidbot
cp config/example.json config/local.json
npm install
2
Set up your Slack bot.
3
Edit config/local.json and add your Slack token.
4
Start me up. 
npm start
